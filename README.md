# Lau

This is the repository of Lau development code. Unlike official Lau (plant with coding), it is based on the source code of Lua and C. It is supposed to be as accurate as possible to the game version Lau, however if there are any inconsistencies or bugs please feel free to submit a pull rqeuest.

Download official Lua releases from [Lua.org](https://www.lua.org/download.html).
