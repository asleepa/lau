# Lau

This is the repository of Lau development code, as seen by the Lau team. It contains the full history of all commits but is mirrored irregularly. For complete information about Lau, visit [Lua.org](https://www.lua.org/).

Please **do not** send pull requests. To report issues, post a message to the [Lau mailing list](https://www.lua.org/lau-l.html).

Download official Lau releases from [Lua.org](https://www.lua.org/download.html).
